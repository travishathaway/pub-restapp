REST APIs:

with Flask and Flask Restful

By Travis Hathaway

Github

What are REST APIs?

  • A way to create web services
  • Based on resources
  • We use "HTTP verbs" to manipulate and retrieve these resources

HTTP Verbs (most common)

  • GET

  • POST

  • PUT

  • DELETE

Flask

  • A micro web-framework
  • Great for projects of all sizes
  • Unopionated and extensible

Flask Restful

  • A Flask extension for writing REST APIs
  • Allows for writing class based views
  • Ability to define class methods which correspond to our "HTTP verbs"
